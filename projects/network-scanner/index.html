Python Network Scanner

This project implements a basic network scanner in Python, leveraging powerful libraries to perform host discovery and port scanning. It's designed for educational purposes to understand network reconnaissance techniques used in ethical hacking.

Features:

  • **Host Discovery:** Identifies active hosts within a specified IP range.
  • **Port Scanning:** Scans for open TCP ports on discovered hosts.
  • **Service Version Detection:** (Optional / Future Feature) Attempts to identify service versions running on open ports.
  • **User-friendly Interface:** Command-line arguments for easy usage.

Technologies Used:

  • Python 3
  • scapy (for crafting and sending network packets)
  • socket (for basic TCP/UDP connections)

How to Use:

To run this scanner, navigate to the project directory and execute the script with a target IP range:

python network_scanner.py 192.168.1.0/24

Example Output:

kali@kali:~$ python network_scanner.py 192.168.1.0/24
    Scanning 192.168.1.0/24...
    Host 192.168.1.1 is UP
        Port 80/tcp is OPEN (http)
        Port 443/tcp is OPEN (https)
    Host 192.168.1.10 is UP
        Port 22/tcp is OPEN (ssh)
    ...
    Scan Complete.
