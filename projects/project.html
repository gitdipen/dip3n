Network Scanner (Python)

This project is a command-line network scanning tool developed in Python. It's designed to help identify active hosts and open ports on a given network segment, simulating initial reconnaissance phases in cybersecurity assessments.

Features:

  • **Host Discovery:** Uses ARP requests to find active devices on the local network.
  • **Port Scanning:** Scans for open TCP ports on target hosts.
  • **Customizable Range:** Allows users to specify IP ranges and port ranges.
  • **Simple CLI:** Easy-to-use command-line interface for quick scans.

Technical Details:

The tool leverages the scapy library for crafting and sending custom network packets (ARP requests, TCP SYN packets) and for parsing responses. It provides a foundational understanding of how network protocols can be manipulated for reconnaissance.

**Key Python Libraries Used:**

  • scapy: For network packet manipulation.
  • argparse: For handling command-line arguments.
  • socket: For basic network operations (though `scapy` handles most of it).

Example Usage (Conceptual):

kali@kali:~$ python network_scanner.py -t 192.168.1.1/24 -p 22,80,443
kali@kali:~$ [+] Scanning 192.168.1.1/24 for hosts and ports...

Host: 192.168.1.1
  Port 22: Open (SSH)
  Port 80: Open (HTTP)

Host: 192.168.1.10
  Port 443: Open (HTTPS)

kali@kali:~$ Scanning complete.
